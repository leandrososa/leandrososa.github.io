Neckselfie

Diseño de interfaz Desarrollo frontend
  • ClienteBulgari (Ficticio)
  • FechaJunio 2018

Neckselfie fue un proyecto ficticio para la joyera italiana Bulgari que pretendía crear una app multiplataforma, para probar diferentes productos mediante realidad aumentada. El cliente accedia virtualmente al catalogo, y luego de testear el articulo podia comprarlo virtualmente o compartirlo en redes sociales.

Se realizó la landing page dentro de la web de Bulgari, que dirigía a los diferentes stores para descargar la aplicación.
